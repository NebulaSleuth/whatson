# Channel art

Required by `manifest`. Drop in PNGs/JPGs at the exact dimensions before
the first store submission.

| File | Dimensions | Format |
|------|------------|--------|
| `icon-focus-hd.png` | 290 × 218 | PNG |
| `icon-focus-sd.png` | 246 × 140 | PNG |
| `splash-hd.jpg`     | 1280 × 720 | JPG |
| `splash-sd.jpg`     | 720 × 480 | JPG |

Sideload installs warn but still succeed if these are missing — fine for
day-to-day dev. Real artwork ships before any user-facing release.
